Task 1: processArray Function

- Takes an array of numbers.
- Returns a new array with even numbers squared and odd numbers tripled.

Task 2: formatArrayStrings Function

- Takes an array of strings and an array of numbers processed by processArray.
- Capitalizes strings if the corresponding number is even; converts to lowercase if odd.

Task 3: createUserProfiles Function

- Takes an array of names and the modified names from Task 2.
- Returns an array of objects with originalName, modifiedName, and id.
